Welcome

Hello and welcome! Today being 15th September 2025, and this page marks the beginning of a student project that reflects the rhythm of university life. Think of it as a little snapshot of my world full of sports, cybersecurity, basketball, music, and the everyday balance between ambition and fun.

About the Project

This assignment focuses on building a clean and functional website using HTML alone. The goal is to understand proper structure, linking between pages, and the importance of accessibility through semantic tags.

Why Semantics Matter

Semantic tags give meaning to content instead of just making it look a certain way. They make the site more accessible, easier for screen readers, and clearer for anyone reading or maintaining the code in the future.

Did You Know?

The very first website ever created in 1991 had no CSS or JavaScript at all. It was made up of only HTML, yet it became the foundation for everything the modern web has grown into today.
